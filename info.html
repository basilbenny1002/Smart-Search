Getting Started

Welcome to the Fastest Search tool! Follow these steps to get started:

  1. Run main.py to index your files initially
  2. Start the search interface with Htmlgui.py
  3. Press Ctrl+Space to open/close the search window
  4. Go to Settings to index your content:
    • Click Index Files for file/folder search
    • Click Index Documents for text content search
    • Click Index Images for image description search
  5. Start searching! Type anything in the search bar
Tip: Enable "Auto-Index New Files" in Settings to automatically index new files as they're created!

Search Types

Normal Search: Search files and folders by name. Ultra-fast tree-based search.

Text Search: Search within document contents. Great for finding specific text in your files.

Image Search: Search images by their descriptions. Uses AI-generated descriptions for semantic search.

Filters: Use the filter buttons to show only specific file types (folders, images, videos, documents, etc.)
